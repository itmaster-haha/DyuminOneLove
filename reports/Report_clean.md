Этот ноутбук предназначен для очистки и стандартизации сырых данных о пассажирах и билетах из файла merged_all_sources.csv.
На выходе формируется очищенный набор данных cleaned_sources.csv, готовый к дальнейшему анализу или загрузке в модель.

Основные возможности

Ноутбук последовательно выполняет:

Загрузка данных

Читает файл merged_all_sources.csv с разделителем ; с помощью pandas.

Очистка структуры таблицы

Удаляет полностью пустые столбцы.

Явно удаляет неиспользуемые поля: Fare, Baggage, Meal.

Нормализация ФИО

Приведение FirstName, LastName, SecondName к верхнему регистру.

Замена буквы Y на I для унификации написания имен.

Нормализация городов

Приведение ArrivalCity и DepartCity к верхнему регистру.

Стандартизация дат

Приведение даты вылета DepartDate к формату ДД.ММ.ГГГГ из различных форматов (включая числа и строки с /).

Приведение даты рождения PassengerBirthDate к формату ДД.ММ.ГГГГ.

Очистка номеров билетов

Замена значения "NOTPRESENTED" в TicketNumber на None (пустое значение).

Удаление дублей и агрегация

Удаление дублирующихся строк.

Группировка по ['TicketNumber', 'FirstName', 'LastName'] и выбор первого непустого значения по другим столбцам (объединение записей об одном и том же пассажире/билете).

Сохранение результата

Запись очищенных данных в файл cleaned_sources.csv с разделителем ;.
